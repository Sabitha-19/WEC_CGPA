🎓 WEC CGPA Tracker

📌 Objective
To build a web-based application that helps students calculate their GPA and CGPA easily using a weighted grade point system.

🚀 Features
Stream and department selection
Semester-wise GPA calculation
Automatic CGPA calculation
Grade reference guide
Simple and user-friendly interface
Responsive design

🛠 Technologies Used
HTML
CSS
JavaScript
Git & GitHub
GitHub Pages (for deployment)
